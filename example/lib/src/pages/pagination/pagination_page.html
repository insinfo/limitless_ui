Pagination options
Tamanhos e alinhamentos seguindo o mesmo padrão do Limitless.
Bordered sizes

A versão padrão suporta `lg`, default e `sm` sem precisar de classes extras fora da API do componente.

Flat sizes

A combinação `variant="flat"` com tamanhos diferentes resolve cenários de páginas compactas e cabeçalhos limpos.

Spaced sizes

A variação `spaced` funciona melhor com pills arredondadas e áreas de navegação mais respiradas.

Left alignment

Alinhamento padrão para listas e blocos de conteúdo tradicionais.

Center alignment

Usa a API de alinhamento do componente para centralizar a navegação sem classes extras no template.

Right alignment

Adequado para footers e barras auxiliares em cards mais densos.
