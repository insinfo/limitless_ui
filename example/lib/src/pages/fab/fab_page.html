FAB menu classes
FAB menu styling is driven by CSS classes and data attributes. The table below summarizes the classes used by this AngularDart wrapper while preserving the original Limitless contract.
Class Description
Basic classes
.fab-menu Main wrapper used by the component.
.fab-menu-btn Main circular trigger button.
.fab-menu-inner Inner action list container.
.fab-icon-open / .fab-icon-close Icons rotated and faded by the Limitless CSS depending on menu state.
Directions and positioning
.fab-menu-top Menu opens below the trigger.
.fab-menu-bottom Menu opens above the trigger.
.fab-menu-fixed Viewport-fixed FAB used for persistent page actions.
.li-fab--dir-left / .li-fab--dir-right Horizontal extensions added by the AngularDart wrapper.
Visibility and labels
data-fab-toggle="click" Click-to-open behavior.
data-fab-toggle="hover" Hover-to-open behavior.
data-fab-state="open" Applied while the menu is expanded.
data-fab-label Tooltip text for inner actions.
.fab-label-end / .fab-label-light / .fab-label-visible Right-aligned, light, and persistent label modifiers.
